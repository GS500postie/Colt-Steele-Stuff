Block and inline elements

content, margin, padding, border and border-box

Explanations

If the display is block the box will break onto a new line. Padding, margin and border will cause other elements to be pushed away from the box. width and height will be respected. If width is not specified, the box will extend in the inline direction to fill the space available in the container. In most cases the box will extend 100% across the page.

Note the paragraph about margin collapsing. This will be helpful when the margins don't behave as expected.

Notice you can get more granular with border styling. Handy to know just in case.

Notice that there is another styling property called outline which is a line outside the element's border. Outlines DON'T TAKE UP SPACE, so they don't affect the layout of the document in any way. Outlines are good for exposing the elements by injecting a red outline in the code to spot all the boxes.
